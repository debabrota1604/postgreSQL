This file mentions how to setup a postgresSQL DB on ARM based Mac (done in M2 Pro).

Installation:
	brew install postgresql 
	
Start service:
	brew services start postgresql@14
	
Stop service:
	brew services stop postgresql 

Verify service running:
    brew services list

Start prompt:
	psql -d postgres
	

All PSQL commands must be ended with semicolon to be executed.



PostgreSQL commands:--------------------------
    Creating DB: CREATE DATABASE debdb;

    Connecting to DB: \c db_name;

    Quit: /q


Some example logs:--------------------------------
